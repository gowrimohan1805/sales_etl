# Sales Performance Dashboard – Python ETL + Tableau

## Project Overview

This project automates the sales data pipeline using Python and visualizes KPIs in Tableau.

### What it does

- Ingests multiple raw CSV files from `data/raw/`
- Cleans and standardizes sales data (dates, regions, categories)
- Calculates business KPIs:
  - Gross Sales
  - Discount Amount
  - Net Sales
  - Profit
  - Profit Margin %
- Outputs a clean, analytics-ready dataset to `data/processed/sales_clean.csv`
- Tableau dashboard is built on top of `sales_clean.csv` for:
  - Region-wise performance
  - Category-wise performance
  - Monthly trends
  - KPI summary (Sales, Profit, Margin)

## Tech Stack

- Python (pandas, python-dateutil, pyarrow)
- Tableau (for dashboard)
